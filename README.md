# 📊 E-Commerce Sales Analysis

## Overview

This project is an end-to-end exploratory data analysis (EDA) of an e-commerce sales dataset. The goal was to investigate the company's sales performance, identify data quality issues, uncover business insights, and provide recommendations based on the analysis.

The project demonstrates a typical data analytics workflow, including data cleaning, exploratory data analysis, visualization, and interpretation of results.

## Data Cleaning Performed

The dataset was inspected and cleaned by:

- Handling missing values
- Removing duplicate records
- Correcting incorrect data types
- Investigating impossible values
- Detecting outliers
- Checking for inconsistent formatting


## Analysis Performed

The analysis explores questions such as:

- Which region generated the highest sales?
- Which region generated the highest profit?
- Which product category generated the most revenue?
- Which products were returned most frequently?
- Which payment method was used most often?
- Which customer age group spent the most?
- Does a higher discount appear to increase sales?
- Are there any monthly sales trends?

A full list of business questions can be found in **questions.md**.


## Key Insights

Some of the insights discovered include:

- Revenue varied across different regions.
- Certain products generated significantly more sales than others.
- Monthly sales fluctuated throughout the year, with noticeable peaks and declines.
- Return rates differed across products.
- Discount levels did not always correspond to higher sales.
